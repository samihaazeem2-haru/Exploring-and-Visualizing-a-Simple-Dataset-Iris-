# Iris Dataset Exploration

# Objective
The goal of this task is to explore and visualize the Iris dataset using Python.  
We perform data loading, inspection, descriptive statistics, and visualizations using Pandas, Matplotlib, and Seaborn.

---

# Dataset Used
- **Name:** Iris Dataset  
- **Source:** Built-in Seaborn dataset  
- **Samples:** 150  
- **Features:**  
  - sepal_length  
  - sepal_width  
  - petal_length  
  - petal_width  
- **Target:** species (Setosa, Versicolor, Virginica)

---

# Tools & Libraries
- Python  
- Pandas  
- Seaborn  
- Matplotlib  
- Jupyter Notebook

---

# Tasks Performed
Loaded dataset with `sns.load_dataset("iris")`  
Checked shape, column names, and previewed dataset  
Displayed summary statistics using `.info()` and `.describe()`  
Created data visualizations:
- Scatter plots  
- Histograms  
- Box plots
